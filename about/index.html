About Multigolf Über Multigolf

Multigolf is a fun and unique tabletop game that doesn't require any preparation, sets or extra materials. By using the participants phones as the game table, people are forced to focus and able to build a tangible course that intuitively makes sense. Participants don't just play a minigolf game on some connected phones; they also construct their own levels in real life. Multigolf ist ein spaßiges und einzigartiges Brettspiel welches keine Vorbereitung, keine extra Materialien und keine gekauften Inhalte benötigt. Indem die Geräte der Spieler als Brett benutzt werden, werden alle dazu gezwungen, sich voll auf das Spiel zu konzentrieren. Man spielt nicht nur irgend ein Online-Minigolf Spiel zusammen, man spielt ein physikalisches Brettspiel bei dem man eigene Kurse und Level bauen muss.

How the Game works Wie das Spiel funktioniert

Every Multigolf game starts with connecting phones to the host device, which is simply the first device to start a game, which the others can join. Then a course is built using the phones in real life, connecting them by drawing lines between them and thus creating a fun and exciting minigolf course that may contain obstacles and other fun features. Afterwards, depending on the chosen game-mode, you will be able to shoot balls between phones into holes! The phones are connected wirelessly and provide a seamless interface that crosses phone borders and creates a fun and satisfying look and experience! Jedes Minigolf Spiel startet damit, dass sich alle teilnehmenden Geräte mit dem Server-Gerät verbinden müssen. Das Server-Gerät ist das Gerät, auf dem das Spiel gestartet wird. Danach wird ein Kurs mit den Geräten gebaut, bei dem Geräte miteinander verbunden werden und Hindernisse, Start-Punkt und Löcher platziert werden können. Je nach ausgewähltem Spielmodus werden dann Bälle irgendwie in die Löcher geschossen! Die Verbindung zwischen den Handys erfolgt kabellos und funktioniert in den meisten Fällen ohne eine Verbindung über externe Server, sondern nur von Handy zu Handy direkt (Peer-To-Peer). Dies ermöglicht superschnelle Verbindungen die ermöglichen, dass Bälle fast ohne Merkbare Probleme von Display zu Display flitzen!

Development and Hosting Entwicklung und Hosting

Multigolf started as a Hackathon project. However, it's now rebuilt and developed by me, Noel Friedrich! The game itself doesn't require much compute power – there is no central golfing server where games are hosted: everything revolves around the host device which connects directly to the players phones via mostly peer-to-peer connections. This enables fast communication between devices (which is important for satisfying screen-to-screen balls) and reduces the cost of running this site dramatically! Multigolf ist als Hackathon Projekt am Trinity College in Dublin gestartet. Es ist nun jedoch neu gebaut und entwickelt von mir, Noel Friedrich! Das Spiel an sich benötigt keine großen Server im Hintergrund, weil die Spiele von den Geräten selbst gehostet werden (deshalb ist es auch nötig, ein Server-Gerät zu haben). Manche Handys/Geräte (besonders oft iPhones) wollen aber keine Peer-To-Peer Verbindung aufbauen, weshalb manchmal sogenannte "Relay-Server" dazwischengeschaltet werden müssen, über die die Daten fliegen. Da aber die meisten Daten zwischen den Handys direkt fliegen kostet das Hosten dieses Spiels nicht viel Geld und kann von mir privat betrieben werden!

I appreciate any help and feedback that I can get. Multigolf bekommt stetig neue Updates und Bugfixes. Es ist nochimmer in einer ziemlich früher Phase der Entwicklung aber schon völlig funktional und spielbar. Ich bin immer bereit für neue Ideen und Feedback und immer erreichbar, wenn es Probleme mit dem Spiel gibt. Ich freue mich über jede Hilfe und Nachricht, die ich bekomme!
