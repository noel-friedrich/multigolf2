FAQ and Help FAQ und Hilfe

Below are some Frequently Asked Questions. If you have any other questions, feel free to contact me here!

Unten sind ein paar Häufig gestellte Fragen (FAQs). Wenn andere Fragen bestehen bin ich über das Kontaktformular erreichbar!

Can I use the host device for the course?
No. The host phone displays useful information for the game, such as status, scores, and whose turn it is, and cannot participate as a device in the Multigolf course. Thus, when you're playing with 5 phones, only 4 can be used for the course itself while one must play host.
Kann das Server-Gerät für den Kurs benutzt werden?
Nein. Das Server-Gerät zeigt wichtige Informationen, den Spielstatus und Anleitungen an. Wenn also z.B. insgesamt 5 Geräte verfügbar sind können nur 4 davon für den Kurs benutzt werden. Es eignet sich gut, einen Laptop oder PC als Server-Gerät zu verwenden.
Can I use an iPad/a Tablet/a Laptop?
Yes! You may use any device with a touch screen for the course. The host device does not need to have a touch screen: it makes sense to use a laptop or computer as the host to increase the number of phones to build the course.
Kann ich ein iPad/ein Tablet/einen Laptop benutzen?
Ja! Du kannst jedes Gerät mit Touch-Screen als Kurs benutzen. Wenn du ein Gerät ohne Touch-Screen benutzen willst kannst du es stattdessen als Server-Gerät verwenden.
How many devices to I need to play?
A good Multigolf course is made up of at least 3 devices plus a host device. You can start games with only two devices (one host, one course), but that won't be much fun.
Wieviele Geräte brauche ich zum Spielen?
Ein guter Multigolf Kurs besteht aus mindestens 3 Geräten (und einem Server-Gerät). Es können auch Kurse mit nur einem Gerät gestartet werden, die aber nicht besonders aufregend sind.
Is there a maximum number of devices?
No. You can theoretically add hundreds of phones, although the rounds might take a while to set up and play. The ideal number of devices might be ~6, since a course that is too big gets boring, especially if you're just waiting for others to finish a round.
Gibt es eine maximale Anzahl von Geräten?
Nein. Du kannst theoretisch hunderte von Geräten hinzufügen, jedoch werden die Runden wahrscheinlich zu lange dauern um in irgend einer Hinsicht Spaß zu machen. Die ideale Anzahl an Geräten liegt wahrscheinlich bei etwa ~6 Geräten.
Why can't I build a circular course?
Most phones have much smaller displays than physical sizes (they have bezels around the display). However, the game does not know how large these are. Thus, the shape of the digital course doesn't match the actual physical model of the course; it only approximates it. This makes it nearly impossible to align devices to both form a circular path digitally and physically.
Wieso kann ich einen Kreis-Kurs bauen?
Die meisten Handys haben ein kleines Display als physikalische Größe (sie haben Ränder). Dieser Rand kann vom Spiel nicht mit einberechnet werden, da das Spiel keine Informationen hat, wie dick dieser Rand bei jedem Handy ist. Es kann also kein realitätsgetreues Abbild des Kurses digital repliziert werden, stattdessen wird es nur approximiert. Das führt dazu, dass ein in realität kreisförmiger Kurs (ein Kurs, dessen "Ende" zum "Anfang" verbunden ist) nicht im digitalen Abbild kreisförmig sein wird.
What happens when a device disconnects ingame?
If the host device fails, the game stops. Try to keep it alive! If a device that is part of the course disconnects, you can simply reconnect it mid-game. If it fails permanently (battery dead?), you'll have to restart the game.
Was passiert wenn ein Gerät im Spiel ausgeht oder das Spiel verlässt?
Wenn sich das Server-Gerät abschaltet endet das Spiel. Wenn ein anderes Gerät im Kurs die Verbindung abbricht kann es neu verbunden werden. Wenn es permanent den Kurs verlassen muss, muss ein neues Spiel angefangen werden.
